# Hotkey Android App v1.0.0.639

## 📱 안드로이드 핫키 관리 앱

GPS 기반으로 위치별 핫키를 설정하고 관리할 수 있는 안드로이드 앱입니다.

## ✨ 주요 기능

- 📍 GPS 위치 기반 핫키 설정
- 🗺️ 네이버 지도 연동
- 📝 위치별 메모 기능
- 🔄 실시간 동기화
- 🎯 맞춤형 단축키 관리

## 📋 시스템 요구사항

- **Android**: 10.0 (API 29) 이상
- **프로세서**: ARM 기반 (ARM64 권장)
- **메모리**: 최소 2GB RAM
- **저장공간**: 약 100MB

## 🔧 설치 방법

1. APK 파일을 다운로드합니다
2. Android 설정에서 "알 수 없는 출처" 허용
3. APK 파일을 실행하여 설치

## 🚀 새로운 기능 (v1.0.0.639)

- 초기 릴리스 버전
- 기본 핫키 관리 기능 구현
- 지도 기반 위치 설정
- 메모 기능 추가
- 성능 최적화 (R8 적용)
